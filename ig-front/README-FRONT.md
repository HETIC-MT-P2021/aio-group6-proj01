# Routes Front-end

These routes let you get categories, and images.

## 1) Routes home
``` /home ```
## 2) Routes categories
### Get categories
``` /categories ```
### Get a single category
``` /categories/{id} ```
### Add a category
``` /categories/{id}/new ```
### Update a category 
``` /categories/{id}/edit ```

## 2) Routes images
### Get images
``` /images ```
### Get an image
``` /images/{id} ```
### Add an image
``` /images/{id}/new ```
### Update an image
``` /images/{id}/edit ```
